### ✅ 3. Health Scoring System (The Innovation)

**Three-Layer Architecture:**

```
Layer 1: Isolation Forest (General ML)
├─ Unsupervised anomaly detection
├─ Trained on Kaggle system metrics
├─ Produces anomaly score (0-100)
└─ Completely independent of user data

Layer 2: Device Configuration (User Context)
├─ Device model (display only)
├─ Hardware specs (RAM, storage)
├─ Device age (purchase date)
├─ Usage pattern (hours/day)
└─ Stored separately from ML

Layer 3: Health Scoring (Personalization)
├─ Formula: Score = Anomaly × AgeFactor × UsageFactor
├─ Age Factor: 0.85-1.0 (device degradation)
├─ Usage Factor: 0.8-1.0 (expected stress)
├─ Clamped to 30-95 range
└─ Produces final health score + risk level
```

**Key Metrics:**
- Health Score: 0-100 numeric
- Risk Level: Low (≥75) | Medium (50-74) | High (<50)
- Transparent Factors: Shown to users
- Recommendation: Auto-generated based on score
