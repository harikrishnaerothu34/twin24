# TWIN24 Frontend

Professional SaaS frontend for the TWIN24 digital twin platform. This frontend is fully standalone and uses mock data only.

## Tech Stack

- React (Vite)
- JavaScript
- Tailwind CSS
- Recharts

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

## Pages

- Landing
- Login (UI only)
- Dashboard
- Devices
- Alerts

## Notes

- No backend integration is included.
- Designed to be API-ready for future integration.
